offsetWidth = width + padding + border; 当前对象的宽度

offsetHeight = height + padding + border; 当前对象的宽度

clientWidth 获取对象可见内容的宽度 包含padding 不包括边框 不包括滚动条

clientHeight 获取对象可见内容的高度 不包含边框 不包括滚动条

scrollWidth 对象的实际内容的宽度，不包含线的宽度，会随着内容超过可视区而变大；当元素的内容超过元素的宽度 而产生滚动条的时候 scrollWidth > clientWidth 如果元素的内容没有超过元素的宽的 此时 scrollWidth = clientWidth

scrollHeight 获取对象滚动条的高度

offsetLeft 当前对象到其上级左边的距离 与其对应的是 style.left

offsetTop 当前对象到其上级顶部的距离 与其对应的是 style.top

clienLeft 获取对象的border的宽度

clientTop 获取对象的border的高度
